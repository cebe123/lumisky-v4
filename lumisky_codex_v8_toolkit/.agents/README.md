# Lumisky repo-scoped Codex skills

Bu klasör progressive-disclosure için tasarlanmıştır: Codex başlangıçta yalnız skill adı/açıklamasını görür; seçilen skill'in tam talimatını daha sonra yükler.

Kurallar:
- Görev başına 1 ana skill.
- En fazla 1 yardımcı skill.
- Önce kısa `.agents/references/*` dosyası; tam V8 mimari yalnız hedef bölüm için.
- Skill açıklamalarını uzatma; implicit routing açıklamanın ilk kısmına bağlıdır.
- Skill içine genel proje kurallarını kopyalama; kök `AGENTS.md` kaynak-of-truth'tur.
